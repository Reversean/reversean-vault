*Логическое отрицание*, *Логическое "НЕ"*, *Инверсия*

Логическая унарная операция. Результатом являтся суждение, "противоположное" исходному.

**Обозначения:** $\neg$, $\overline{A}$, $NOT$

**Таблица истинности для бинарной конъюнкции:**

| $a$ | $\neg a$ |
| --- | -------- |
| 0   | 1        |
| 1   | 0        |
