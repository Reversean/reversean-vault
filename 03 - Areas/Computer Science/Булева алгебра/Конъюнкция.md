*Логическое умножение*, *Логическое "И"*, *"И"*

Логическая операция. Имеет бинарную/тернрную/n-арную формы.

**Обозначения:** $\&$, $\&\&$, $\land$, $AND$

**Таблица истинности для бинарной конъюнкции:**

| $a$ | $b$ | $a \land b$ |
| --- | --- | ----------- |
| 0   | 0   | 0           |
| 1   | 0   | 0           |
| 0   | 1   | 0           |
| 1   | 1   | 1           |
